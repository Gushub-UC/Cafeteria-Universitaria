# Cafetería UDC — Prototipo de Landing (React)

Proyecto mínimo en React (Vite) que sirve como landing page y prototipo para un sistema de pedidos de la cafetería.

Archivos importantes:
- `src/data/users.json` — Usuarios / Clientes
- `src/data/products.json` — Productos
- `src/data/categories.json` — Categorías
- `src/data/pedidos.json` — Pedidos
- `src/data/detallePedidos.json` — Detalle de pedidos
- `src/data/pagos.json` — Pagos

Instrucciones rápidas:

1. Instalar dependencias:

```bash
npm install
```

2. Ejecutar servidor de desarrollo:

```bash
npm run dev
```

El prototipo es estático y carga los `.json` para mostrar datos de ejemplo.
# Cafeteria-Universitaria
Desarrollar y publicar un prototipo funcional de una aplicación que permita a los estudiantes y personal universitario realizar pedidos en la cafetería del campus desde sus dispositivos móviles, reduciendo tiempos de espera y mejorando la experiencia de atención.
